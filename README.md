# Basic Informational Site

A simple Node.js web server project from The Odin Project curriculum.

## What This Project Does

This is a basic web server built with Node.js that serves static HTML pages. It demonstrates the fundamentals of:

- Creating a Node.js HTTP server
- Handling different routes/URLs
- Serving HTML files
- Implementing a 404 error page

## Files

- `server.js` - The Node.js server code
- `index.html` - Home page
- `about.html` - About page
- `contact-me.html` - Contact page
- `404.html` - Error page for non-existent routes

## How to Run

1. Run the server:
   ```
   node server.js
   ```
2. Open your browser and visit:
   - `http://localhost:8080` - Home page
   - `http://localhost:8080/about` - About page
   - `http://localhost:8080/contact-me` - Contact page

## Learning Goals

This project helped me understand:

- How Node.js servers work
- Routing requests to different pages
- Reading and serving files
- Basic error handling
